| comando                                                     | funcion                            |
|-------------------------------------------------------------|------------------------------------|
| node -v                                                     | version de Node js                 |
| npm install -g @angular/cli                                 | instalar ultima version de angular |
| ng version                                                  | version instalada de angular/cli   |
| cls                                                         | para limpiar la consola            |
| git init                                                    | iniciar repositorio                |
| git status                                                  | verificar estado de mis archivos   |
| git add .                                                   | agregamos archivos al git          |
| git commit -m "V1_Angular"                                  | sellamos los archivos para enviar  |
| git remote add origin https://github.com/SebasG789/xxxx     | enlazar repo                       |
| git remote -v                                               | verificar repositorio remoto       |
| git push                                                    | para enviar al repo remoto         |
| ng new mi-primer-angular --standalone --routing --style=css | crear proyecto                     |
